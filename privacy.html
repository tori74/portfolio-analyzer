1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сервиса анализа инвестиционных портфелей (далее — Сервис).

Оператор персональных данных: [ФИО / название ИП / ООО], ИНН: [ваш ИНН].

2. Какие данные мы собираем

При использовании Сервиса могут быть собраны следующие данные:

— Имя (для персонализации отчёта);

— Данные инвестиционного портфеля (состав активов, их стоимость) из загруженных файлов;

— Параметры инвестирования (риск-профиль, горизонт, статус госслужащего).

3. Цели обработки данных

Персональные данные используются исключительно для:

— Формирования персонального анализа инвестиционного портфеля;

— Генерации PDF-отчёта с рекомендациями.

4. Хранение и защита данных

Загруженные файлы и данные портфеля обрабатываются в момент запроса и не сохраняются на серверах после генерации отчёта. Временные файлы удаляются автоматически.

5. Передача данных третьим лицам

Персональные данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации.

Для генерации аналитических текстов используется API сервиса Anthropic (Claude). При этом передаются только обезличенные данные портфеля (состав активов, доли), без персональных данных пользователя.
